# SQL Practice - Northwind

Este repositório reúne exercícios de SQL que resolvi durante meus estudos em Dados, utilizando o banco de dados Northwind em PostgreSQL.

## 🎯 Objetivo

Fortalecer minha base em SQL por meio de prática com cenários reais de negócio, envolvendo análise de clientes, pedidos, faturamento e logística.

## 🧠 Conteúdo abordado

Os exercícios incluem:

- SELECT e filtros (WHERE)
- ORDER BY
- GROUP BY e agregações
- JOINs entre múltiplas tabelas
- subqueries
- CTEs (Common Table Expressions)
- funções de janela (window functions)

## 📊 Exemplos de análises realizadas

- clientes por região
- pedidos com frete acima da média
- produtos mais caros
- faturamento por pedido
- ranking de clientes por receita
- faturamento acumulado ao longo do tempo
- clientes com maior volume de pedidos
- análise de relacionamento entre tabelas

## 📁 Estrutura

Os exercícios estão organizados em um arquivo `.sql`, contendo consultas e desafios resolvidos.
